# lightning-pong

## Install

```shell script
npm install
npm run tsc
```

## Usage

### Connect to LND

```shell script
node ./index connect 027455aef8453d92f4706b560b61527cc217ddf14da41770e8ed6607190a1851b8@127.0.0.1:9735
```

### Listen for LND connections

```shell script
node ./index listen -p 1337
```
